## 网上课堂项目
采用SpringBoot搭建的几个模块，实现各模块的调用

本项目采用前后端分离的方案，以实现主要功能，其中包括（后台管理系统：讲师的CRUD，讲师头像，列表，详细信息的的显示及更改；课程信息通过EXECL表格导入，课程相关发布，课程视频上传至阿里云，通过图表统计生成平台相关数据；）

###后端选型

环境：JDK1.8、IDEA
- Springboot
- MySQL
- MyBatisPlus
- SpringCloud微服务之间调用
- Nginx反向代理
- redis缓存
- 阿里云OSS
- 阿里云视频点播
### 前端选型

- vue
- element-ui
- vue-template-admin
- Nuxt
